# CHANGELOG

这里记录代码修改记录，供协同时更新。Codex启动后请先阅读此文件了解最新变化。

 ## 2026-03-31

### 修复
- **管理员看板总是显示第一个创建的家庭，没有按当前用户过滤 → 已修复，现在只显示当前登录用户自己创建的家庭
- **受邀新用户没有注册入口** → 新增 `/join/[token]/set-password` 页面，未登录接受邀请后直接跳这里设置密码完成注册，不用跳登录页
- **练习页面中英文重复显示** → 已修复，现在只显示当前语言标题和描述
- **按钮对比度看不清** → 全局优化文字对比度，现在所有按钮文字都清晰

### 新增
- 新增 `accept-with-password` API，一步完成创建用户+接受邀请+自动登录
- 新增 `acceptInviteAfterCreateUser` 函数在 `invite-repository.ts`
- 新增 **DSE 口语模拟考页面** `/speaking/mock`，完整考试流程：准备倒计时 → 录音 → 评估
- 新增 **语法练习模块** → 选择题模式，按难度推荐，做完即时显示对错，结果存储到数据库
- 新增 **写作模块完整开发** → 支持语音输入直接口述作文，自动转文字

### 改进
- **口语练习流程重构**：
  - 首页分两个入口：模式练习 + DSE模拟考
  - 模式练习每天限定 **4句短语** → 符合要求"先做4句练习评估通过，再进入模拟考"
  - 全程直接点击录音，不需要上传文件 → 改善用户体验
- **听力模块重构**：
  - 支持嵌入 YouTube 视频，不需要自己托管音频文件
  - 降低了添加新听力练习题的成本，可以直接利用 YouTube 现成内容
  - 优化隐私保护：使用 `youtube-nocookie.com` 域名减少第三方跟踪
  - 添加懒加载：提升页面加载速度
- **添加**：
  - Zod 环境变量类型验证 → 启动时立即发现配置错误，不会运行到一半报错
  - 完善的健康检查端点 `/api/health` → 部署监控可以直接检查数据库连接状态
  - `FamilyInvite` 添加唯一约束 `(familyId, email)` → 防止同一个邮箱在同一个家庭收到多个邀请，保证数据一致性
