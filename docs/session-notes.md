# Session Notes

## Product direction

- Product name: `DSE_Study`
- Primary users: one F3 learner, one P5 learner, one parent admin user
- UI language: English-first interface
- Mobile-first web app
- Daily study target: 40-60 minutes, 6 days per week

## Learning design

- DSE-focused but usable for different levels
- Initial baseline assessment: 30-40 minutes
- Bi-weekly review cycle
- Reference Level + Internal Band progression model
- Vocabulary Loop with spaced review and pronunciation practice
- Speaking Phrase Loop with listen, shadow, replay, substitute, and use-it steps

## Permissions

- Mom: family admin, can view all accounts
- Student users: can view only their own progress
